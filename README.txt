libframegen - Библиотека для отправки/приема пакетов библиотекой librfc2544.

Компиляция:
  make - сборка libframegen.so и libframegen.a
  make install - установка в папку, заданную параметром PREFIX

Описание:
  Вызовы всех функций libframegen происходят из libfrc2544. Программа,
  связываемая с libframegen, должна экспотрировать строки rx_ifname и
  tx_ifname с именами интерфейсов для приема/отправки пакетов(см. export.h).

  Если от ядра приходят таймстампы SO_TIMESTAMPING, используются они, иначе
  снимаются через clock_gettime прямо перед отправкой и после получения
  пакетов.

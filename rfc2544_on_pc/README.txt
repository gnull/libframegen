rfc2544_on_pc - программа для тестирования сети по rfc2544.

Компиляция:
  make

Описание:
  Программа получает в качестве параметров командной строки (их 20+)
  все поля структуры rfc2544_settings_t. Если параметр не задан,
  по-умолчанию используется 0 (кроме нескольких полей header_cfg_t).

  Для удобства пример параметров вынесен в default.conf. Для запуска
  программы с этими параметрами служит скрипт ./rfc2544-default.sh.
  Ему можно передать дополнительные параметры, которые перезапишут
  заданные в default.conf.

  Подробнее см. ./rfc2544_on_pc --help
